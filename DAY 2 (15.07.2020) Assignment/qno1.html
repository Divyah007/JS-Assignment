Write a js code which takes input from the user and logs it in the console.

WINDOWS PROMPT()

  • The prompt() method displays a dialog box that prompts the visitor for input.
  • A prompt box is often used if you want the user to input a value before entering a page.
  • When a prompt box pops up, the user will have to click either "OK" or "Cancel" to proceed after entering an input value
  • Do not overuse this method, as it prevents the user from accessing other parts of the page until the box is closed.
  • The prompt() method returns the input value if the user clicks "OK" .
  • If the user clicks "cancel" the method returns null.
  • Example: let age=prompt("What is your age?"); console.log(age);
