CONSOLE OBJECT:

  • The Console object provides access to the web browser's debugging console.
  • For example, consider Google Chrome web browser, to enter into Chrome's console
  • press F12 key (or)
  • right click and select Inspect (or)
  • click on the Customize and control Google Chrome(three vertical dots which is present on the top right side)--->More tools--->Developer tools

METHODS OF CONSOLE OBJECT:

The console object provides several different methods, they are as follows,

  • console.log()
  • console.error()
  • console.warn()
  • console.table()
  • console.assert()
  • console.time()
  • console.timeEnd()
  • console.group()
  • console.groupEnd()
  • console.clear()

console.log()

  • The console.log() method writes a message to the console.
  • Example : console.log("Hello world!");

console.error()

  • The console.error() method writes an error message to the console.
  • Example: console.error("This is an error!");

console.warn()

  • The console.warn() method writes a warning to the console.
  • Example: console.warn("This is a warning!");

console.table()

  • The console.table() method writes a table in the console view. The first parameter is required, and must be either an object, or an array, containing data to fill the table.
  • Example:console.table(["Dia", "Sara", "Tia"]);

console.time()

  • The console.time() method is used to starts a timer and can track how long an operation takes.
  • Example: console.time("Hello World");

console.timeEnd()

  • The console.timeEnd() method is used to stop a timer which was started by console.time()
  • Example: console.time("Hello World");

console.group()

  • The console.group() method indicates the start of a message group. All messages will from now on be written inside this group.
  • Example: console.log("Hello world!"); console.group(); console.log("Hello again, this time inside a group!");

console.groupEnd()

  • The console.groupEnd() method indicates the end of a message group.
  • Example: console.log("Hello world!"); console.group(); console.log("Hello again, this time inside a group!"); console.groupEnd(); console.log("and we are back.");

console.clear()

  • The console.clear() method is used to clear the console.
  • Example: console.clear();
